# Home Lab Portfolio

Ce dépôt contiendra mes projets réalisés dans le cadre de ma montée en compétences en systèmes, réseaux, cloud et cybersécurité.

## Objectifs
- Apprendre Linux et administration système
- Comprendre les réseaux et la sécurité
- Découvrir le Cloud et le déploiement d’infrastructure
- Créer un portfolio montrant mes compétences techniques

## Compétences visées
- Linux (Ubuntu Server)
- Réseaux (TCP/IP, VLAN, routage)
- Sécurité (firewall, SSH hardening)
- Cloud (AWS & Azure basics)
- Automatisation (Bash, PowerShell)

## Projets prévus
1. Installation et configuration Ubuntu Server dans une VM
2. Serveur web Apache/Nginx
3. Serveur DNS local
4. VPN WireGuard
5. Firewall UFW + règles sécurisées
6. Scripts Bash d’automatisation
7. Déploiement EC2 sur AWS avec Terraform
